fccBot

Meet fccBot - the Slack bot for freeCodeCampers

Get information on freeCodeCamp challenges, right in Slack!

Search by challenge or certificate

View user stories for challenges

Get links to information and resources

Learning to code? We make searches easy, so that you can focus on your goals.

Just type /fccbot to get started.

Type /fccbot [certificate] to see the challenges for your next freeCodeCamp certificate, then select a challenge to view details.

In a rush? The command /fccbot [project name] takes you straight to information about your current project.

Add fccBot to your Slack team.

The fccBot was created by members of the Chingu Cohorts - student programmers learning together.

This project has not been endorsed by freeCodeCamp - but we like to think they'd approve.
