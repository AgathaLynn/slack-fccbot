About

fccBot was created by members of the Chingu Cohorts - student programmers learning together. The fccBot is not part of the freeCodeCamp project and was not created or endorsed by freeCodeCamp - but we like to think they'd approve.

To learn more about fccBot, check out the GitHub repository for the fccBot project.
